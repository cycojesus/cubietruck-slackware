Installation steps Arch Linux for Cubietruck
---------------------------------------------------------

use a debian based Linux Distribution!

open your preferred shell
sudo apt-get -y install git
cd ~
git clone https://bitbucket.org/mziegler08/cubietruck-arch
cd cubietruck-arch
chmod +x build.sh
sudo ./build.sh
